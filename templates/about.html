About Retail Predictions

This web app allows the user to input information about a store-department and receive a prediction for weekly sales. The app is built using Flask and the machine learning model is built in Python using the XGBoost learning API.


After inputting values for each variable and hitting predict, you will receive both a prediction as well as graphic indicating which variables most influenced the model. Try experimenting with different values like Week=51 (right during the holidays) and Department 92 (which has very high sales volume) to see how it impacts the prediction.

More information about the project, including all of the code for the model, can be found in the project repository:

https://github.com/jamesdinardo/Retail-Forecasting
